#### Gruppe 2 Frontend
### What this App does:
This app is the Frontend for the Project MrJacooo/Gruppe2-VP-BE.
This app allows you to created ToDo's and assign them to different Persons.
It was realized as a schoolproject at the BBW.

### Functionality
✔ Navigation
✔ Add Todo
✔ Edit Todo
✔ Delete Todo
X Add Person
✔ Edit Person
✔ Delete Person
